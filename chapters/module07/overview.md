# Module 7: Advanced SQL

## Overview

Take your SQL skills to the next level! This week covers **JOINs** to combine data from multiple tables, **subqueries** for complex filtering, and **aggregation** for summarizing data.

## Learning Objectives

By the end of this module, you will be able to:

- Join multiple tables using INNER, LEFT, RIGHT, and FULL joins
- Write subqueries in SELECT, FROM, and WHERE clauses
- Use aggregate functions (COUNT, SUM, AVG, MIN, MAX)
- Group and filter aggregated data with GROUP BY and HAVING

## Topics Covered

1. **JOINs** - Combining tables
2. **Subqueries** - Nested queries
3. **Aggregation** - Summarizing data
