# Module 8: Advanced SQL II

## Overview

This week focuses on **database performance and reliability**. You'll learn about views, indexes, transactions, and stored procedures—essential tools for professional database work.

## Learning Objectives

By the end of this module, you will be able to:

- Create and use views
- Design indexes to improve query performance
- Manage transactions with ACID properties
- Create basic stored procedures

## Topics Covered

1. **Views** - Virtual tables
2. **Indexes** - Performance optimization
3. **Transactions** - Data integrity
4. **Stored Procedures** - Reusable code
