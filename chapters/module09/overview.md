# Module 9: Cloud DBs & Open CMS

## Overview

Welcome to the modern database landscape! This week explores **cloud databases** and **NoSQL** alternatives to traditional relational systems.

## Learning Objectives

By the end of this module, you will be able to:

- Compare cloud database services (AWS RDS, Google Cloud SQL, Azure)
- Explain the advantages of cloud-hosted databases
- Identify different types of NoSQL databases
- Choose between relational and NoSQL for given scenarios

## Topics Covered

1. **Cloud Databases**
   - Why move to the cloud?
   - Major providers and services
   
2. **NoSQL Databases**
   - Document, key-value, graph, column-family
   - When to use NoSQL
