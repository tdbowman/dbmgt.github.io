# Module 5: SQL Statements

## Overview

Welcome to SQL! **Structured Query Language** is the standard language for communicating with relational databases. This week, you'll write your first queries to retrieve data.

## Learning Objectives

By the end of this module, you will be able to:

- Write SELECT statements to retrieve data
- Use WHERE to filter results
- Sort data with ORDER BY
- Limit results and remove duplicates

## Topics Covered

1. **Introduction to SQL**
   - SQL history and standards
   - DDL vs DML vs DCL

2. **SELECT Statements**
   - Basic SELECT syntax
   - Selecting specific columns
   - WHERE clause
   - Comparison and logical operators
   - LIKE pattern matching
   - ORDER BY, LIMIT, DISTINCT
