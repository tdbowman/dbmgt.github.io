# Module 14: Security & Privacy

## Overview

Our final module covers the critical topics of **database security** and **privacy**. As a database professional, protecting data is one of your most important responsibilities.

## Learning Objectives

By the end of this module, you will be able to:

- Implement authentication and authorization
- Prevent SQL injection attacks
- Apply encryption basics
- Understand privacy regulations (GDPR, CCPA)

## Topics Covered

1. **Security**
   - Authentication and authorization
   - SQL injection prevention
   - Encryption

2. **Privacy**
   - Data protection regulations
   - Privacy by design
   - Data governance
