# Module 10: Python & PostgreSQL Integration

## Overview

Time to connect your database to code! This week you'll learn to use **Python with PostgreSQL** and build your first **Flask web application**.

## Learning Objectives

By the end of this module, you will be able to:

- Connect Python to PostgreSQL using psycopg2
- Use SQLAlchemy for object-relational mapping
- Build basic Flask routes
- Display database data in web pages

## Topics Covered

1. **Python Database Connectivity**
   - psycopg2 basics
   - SQLAlchemy ORM

2. **Flask Web Framework**
   - Routes and views
   - Templates with Jinja2
   - Database integration
