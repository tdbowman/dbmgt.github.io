# Module 11: Web Application Project

## Overview

An example of a web-based Database Project! This week focuses on an example of a **full Flask + PostgreSQL web application** with CRUD operations.

## Learning Objectives

By the end of this module, you will be able to:

- Explore a web-based CRUD application
- Understand creating forms for data entry
- Consider handling errors gracefully
- Thinking about security best practices

## Topics Covered

1. **Complete CRUD Operations**
2. **Forms and Validation**
3. **Error Handling**
4. **Best Practices**
