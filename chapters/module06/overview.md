# Module 6: Creating & Editing Tables

## Overview

Now that you can query data, it's time to create your own tables! This week focuses on **DDL (Data Definition Language)**â€”the SQL commands for creating and modifying database structures.

## Learning Objectives

By the end of this module, you will be able to:

- Create tables with appropriate data types
- Define primary and foreign keys
- Add constraints (NOT NULL, UNIQUE, CHECK)
- Modify existing tables with ALTER
- Remove tables with DROP

## Topics Covered

1. **CREATE TABLE**
   - Syntax and data types
   - Constraints in DDL
   
2. **ALTER TABLE**
   - Adding/removing columns
   - Modifying constraints

3. **DROP TABLE**
   - Removing tables safely
