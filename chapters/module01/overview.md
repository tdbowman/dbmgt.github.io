# Module 1: Introduction to Databases

## Overview

Welcome to your first module! This week we'll introduce you to the course, get you set up with the necessary tools, and start exploring the basics of database management systems.

## Learning Objectives

By the end of this module, you will be able to:

- Navigate the course Canvas site and locate resources
- Install and configure PostgreSQL on your computer
- Install and use DBeaver for database management
- Define what a database and DBMS are
- Explain the advantages of DBMS over file-based systems
- Identify common database applications in everyday life

   - File systems vs. DBMS

3. **Setting Up Your Environment**
   - Installing PostgreSQL
   - Installing DBeaver
   - Connecting to your first database
   - Basic navigation
