# Additional Resources

A curated collection of resources to support your database learning journey.

## Official Documentation

- [PostgreSQL Documentation](https://www.postgresql.org/docs/) - The official and comprehensive PostgreSQL docs
- [PostgreSQL Tutorial](https://www.postgresqltutorial.com/) - Step-by-step PostgreSQL tutorials
- [DBeaver Documentation](https://dbeaver.com/docs/) - Guide to using the DBeaver database tool

## Online Courses & Tutorials

### Free Resources

- [Khan Academy - Intro to SQL](https://www.khanacademy.org/computing/computer-programming/sql) - Interactive SQL basics
- [SQLZoo](https://sqlzoo.net/) - Interactive SQL tutorials with exercises
- [Mode SQL Tutorial](https://mode.com/sql-tutorial/) - Practical SQL for data analysis
- [W3Schools SQL](https://www.w3schools.com/sql/) - Quick reference and examples

### Interactive Practice

- [SQLBolt](https://sqlbolt.com/) - Learn SQL with interactive exercises
- [HackerRank SQL](https://www.hackerrank.com/domains/sql) - SQL challenges by difficulty
- [LeetCode Database](https://leetcode.com/problemset/database/) - SQL problems for interview prep
- [Exercism SQL Track](https://exercism.org/tracks/sql) - Practice with mentorship
